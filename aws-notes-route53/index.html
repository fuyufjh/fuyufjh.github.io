AWS 学习笔记之 Route53

发表于 2017-04-04 | 更新于 2017-08-19 |

Route53 是 AWS 的域名解析（DNS）服务。

ELB 没有固定的 IPv4 地址，用域名解析过去。

理解 Alias record 和 CNAME record 的区别：Alias 是 AWS 特有的一种解析方式，只能解析到 AWS Resources 上，但是具有 CNAME 不具备的一些特性，例如，解析裸域名。

考试中，如果让你选择 CNAME 或 Alias，大多数情况下都是选 Alias

有以下几种 routing policies：

  • Simple - 简单解析
  • Weighed - 根据设置的权重把流量解析到不同的服务器上，常用于AB测试
  • Latency - 根据用户到各个 region 的延迟，选择最快的解析
  • Failover - 主从备份，当主服务器 fail 时解析到备份服务器
  • Geolocation - 根据用户的地理位置决定解析，例如网站有不同语言
